# Prompt Templates

## Navigator Agent

Given user location, destination, accessibility needs, live density, blocked zones, available elevators, and venue map context, return route steps with ETA, safety rationale, and fallback instructions.

## Emergency Copilot

Classify severity, extract location, identify nearest responder, generate route, create notification copy, and draft a concise incident report. Use calm language and include confidence.

## Crowd Intelligence Agent

Analyze crowd density, motion vectors, gate throughput, CCTV events, WiFi/Bluetooth counts, and match timing. Predict congestion and propose operational actions with risk score.

## Natural-Language Analytics Agent

Answer only from retrieved telemetry and cited records. Include forecast confidence, assumptions, recommended action, and the operational owner.

## Translation Agent

Translate stadium guidance into the user's selected language. Keep messages short, clear, culturally neutral, and usable in noisy environments.
